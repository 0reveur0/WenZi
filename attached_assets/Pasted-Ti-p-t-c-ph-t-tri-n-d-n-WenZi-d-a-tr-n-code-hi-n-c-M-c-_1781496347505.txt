Tiếp tục phát triển dự án WenZi dựa trên code hiện có.

## Mục tiêu

Xây dựng chức năng tra cứu từ điển Hán Việt cơ bản hoạt động hoàn chỉnh.

## Yêu cầu

### Database

Sử dụng SQLite hiện có.

Tạo bảng `dictionary` với cấu trúc:

* id (INTEGER PRIMARY KEY AUTOINCREMENT)
* simplified (TEXT NOT NULL)
* traditional (TEXT)
* pinyin (TEXT)
* vietnamese (TEXT NOT NULL)

Nếu bảng chưa tồn tại thì tự động tạo khi khởi động ứng dụng.

---

## Dữ liệu mẫu

Chèn khoảng 10–20 từ mẫu nếu database đang trống.

Ví dụ:

* 中国 → Trung Quốc
* 学生 → Học sinh
* 老师 → Giáo viên
* 学习 → Học tập
* 谢谢 → Cảm ơn

Bao gồm:

* Giản thể
* Phồn thể
* Pinyin
* Nghĩa tiếng Việt

---

## API

Tạo endpoint tìm kiếm:

```text
GET /api/search?q=<keyword>
```

Yêu cầu:

* Tìm kiếm theo:

  * chữ giản thể
  * chữ phồn thể
  * pinyin
  * nghĩa tiếng Việt
* Không phân biệt chữ hoa/chữ thường đối với pinyin và tiếng Việt.
* Trả về JSON.

Ví dụ:

```json
[
  {
    "simplified": "中国",
    "traditional": "中國",
    "pinyin": "zhōng guó",
    "vietnamese": "Trung Quốc"
  }
]
```

---

## Giao diện

Hoàn thiện trang chủ:

### Thanh tìm kiếm

* Tìm kiếm realtime khi người dùng nhập.
* Debounce khoảng 300ms.
* Hiển thị trạng thái đang tìm kiếm.
* Hiển thị thông báo khi không tìm thấy kết quả.

### Hiển thị kết quả

Mỗi từ hiển thị dưới dạng card gồm:

* Chữ giản thể (nổi bật nhất)
* Chữ phồn thể
* Pinyin
* Nghĩa tiếng Việt

Thiết kế:

* Gọn gàng
* Dễ đọc
* Responsive cho mobile và desktop
* Hỗ trợ dark mode

---

## Xử lý lỗi

* Nếu API lỗi, hiển thị thông báo thân thiện.
* Không để ứng dụng crash.
* Có trạng thái loading và empty state.

---

## Yêu cầu code

* Sử dụng TypeScript.
* Giữ code đơn giản và dễ hiểu.
* Không thêm framework state management.
* Không thêm package không cần thiết.
* Không thay đổi cấu trúc dự án hiện tại.
* Nếu sửa file, luôn cung cấp nội dung hoàn chỉnh của file.
* Sau khi hoàn thành, giải thích ngắn gọn luồng hoạt động từ: người dùng nhập từ khóa → API → SQLite → hiển thị kết quả.
