Hãy đóng vai một chuyên gia Frontend Senior và AI Coding Assistant. Tôi cần bạn thực hiện "Prompt 4: Xây dựng nền tảng dữ liệu cho WenZi". Mục tiêu của bước này là chuyển đổi WenZi từ một bản demo (dữ liệu hard-code) thành một từ điển thực sự với khả năng tra cứu hơn 125.000 từ (dựa trên CC-CEDICT).

Vì ứng dụng được deploy tĩnh hoàn toàn trên GitHub Pages (Serverless / No Backend), chúng ta cần giải quyết bài toán: Làm sao để tìm kiếm nhanh trên 125.000 từ ngay tại trình duyệt mà không làm sập bộ nhớ hoặc đơ UI.

Hãy thực hiện các bước sau:

### 1. Thiết kế cấu trúc dữ liệu (Data Architecture)
- Thiết kế cấu trúc một đối tượng từ vựng (Word Object) chuẩn bao gồm: `simplified` (giản thể), `traditional` (phồn thể), `pinyin` (có dấu), `pinyin_raw` (không dấu để dễ tìm kiếm), `han_viet` (âm Hán Việt), `definition` (nghĩa tiếng Việt và tiếng Anh).
- Đề xuất giải pháp phân tách dữ liệu (Data Splitting/Chunking) theo chữ cái đầu của Pinyin hoặc theo độ phổ biến (ví dụ: chia nhỏ thành các file `.json` khoảng 1-2MB) để tối ưu lazy-loading khi user gõ tìm kiếm.

### 2. Cấu trúc thư mục mới
Hướng dẫn tôi tạo cấu trúc thư mục chứa dữ liệu trong `src/`, ví dụ:
- `src/data/chunks/` (chứa các file dữ liệu đã chia nhỏ)
- `src/services/searchService.js` (xử lý logic tìm kiếm)

### 3. Tích hợp công cụ tìm kiếm Client-Side (Client-side Search Engine)
- Viết code thiết lập một thư viện tìm kiếm client-side tối ưu (khuyến khích dùng Fuse.js hoặc FlexSearch để tìm kiếm mờ - fuzzy search).
- Bộ lọc tìm kiếm phải hỗ trợ:
  + Tìm theo chữ Hán (Giản thể / Phồn thể).
  + Tìm theo Pinyin (có dấu `xuéxí` hoặc không dấu `xuexi`).
  + Tìm theo Nghĩa / Âm Hán Việt.

### 4. Code Component tìm kiếm & Hiển thị kết quả (Search & Virtual List)
- Cập nhật Component tìm kiếm chính (`SearchBar` và `ResultList`).
- Sử dụng kỹ thuật Virtual Scroll (Ví dụ: `react-window` hoặc tự code) để đảm bảo khi kết quả trả về hàng ngàn từ, giao diện vẫn mượt mà, không bị lag.
- Thêm hiệu ứng Loading trạng thái khi ứng dụng đang tải dữ liệu chunk từ file JSON.

Hãy giải thích ngắn gọn tư duy giải quyết, sau đó cung cấp toàn bộ mã nguồn chi tiết, sạch sẽ (Clean Code) sử dụng React (Vite) + Tailwind CSS để tôi áp dụng vào dự án WenZi ngay lập tức.
